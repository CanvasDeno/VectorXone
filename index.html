Download for Self-Hosting (Offline Use)

You can download the editor as a single `.html` file for privacy or offline use. Use the curl command below.

⚠️ **Prerequisite:** This command requires the curl utility. If it's not installed on your system, please install it using your operating system's package manager first (e.g., sudo apt install curl on Debian/Ubuntu). 

curl -i https://canvasdeno.github.io/VectorXone/editor.html
